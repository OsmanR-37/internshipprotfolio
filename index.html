My Projects

Project 1

This is my first project. It's a game called Superhero tycoon, in which players select 1 of 4 buildings, each one being based off a different superhero, which they then make a building based off of. This project was created on the 30th of May 2023, and was last updated 1 year ago. It is no longer being updated, due to me working on other projects.

Project 2

This is my second project. It is a game called 2 Player Obby. The game was developed to help promote teamwork with others. In this game, 2 players are required to complete an obstacle course. However, for a player to complete a stage, their teammate must step on a button, for the path to appear. This game was created on the 19th of February 2024, and was last updated 1 year ago. The game has not been updated since them, but I am planning to release an update soon, in which I revamp the winners section.

Project 3

This is my third project, and it's the biggest one by far. It's a game called A camping trip. This game is a multiplayer story game in which you and a group of other people go on a camping trip. During the trip, a lot of paranormal events occur, which players have to survive in order to win. The game has 2 endings, both of which are accessed through a vote. This game was created on the 31st of July 2023, and was last updated 8 months ago. It is currently in the testing stage, and will continue to be updated, adding new features such as a minimap, a bossfight, and even more.
